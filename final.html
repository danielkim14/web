Plagiarism Detector

Daniel Kim Period 9

  • Project Description
    • This is a program that detects plagarism in two different pieces of text.
  • Instructions
    • Open the python program and enter one file from your computer where it says "text1" and another file you want to compare it with in "text2". Click Enter and then the results will appear. It will tell you how similar the two pieces of texts are.
  • How it works
    • Our code involves the usage of the difflib module from SequenceMatcher. This module helps with comparing sequences and files, which will be a foundation for our program. First, the user will upload two text in the dictionary (we provided a sample text for practice) and then the files will be read. When the files are read the similarities and differences are computed. Sequencematcher matches the two sequences from text1 and text2. Lastly, the results are produced as a statement telling you how similar the two texts are.
